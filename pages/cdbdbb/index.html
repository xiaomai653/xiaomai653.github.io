Docker安装Tomcat

Apache Tomcat (opens new window)

# docker安装tomcat

# 1.获取镜像列表

docker search tomcat

# 2.拉取镜像

docker pull tomcat

# 3.运行

docker run -d --name tomcat -p 8080:8080 -v /home/docker/tomcat/webapps:/usr/local/tomcat/webapps/ -v /home/docker/tomcat/conf/server.xml:/usr/local/tomcat/conf/server.xml -v /home/docker/tomcat/logs:/usr/local/tomcat/logs/ tomcat:latest

-d：后台运行
-p：将容器内部端口向外映射
--name：命名容器名称
-v：将容器内数据文件夹或者日志、配置等文件夹挂载到宿主机指定目录

# docker compose安装tomcat

# 1.创建yml文件

docker-compose-tomcat.yml

version: '3.3'
services:
  tomcat:
    image: tomcat
    container_name: tomcat
    ports:
      - "8080:8080"
    environment:
      TZ: Asia/Shanghai
    privileged: true
    volumes:
      - /home/docker/tomcat/webapps:/usr/local/tomcat/webapps/
      - /home/docker/tomcat/conf/server.xml:/usr/local/tomcat/conf/server.xml
      - /home/docker/tomcat/logs:/usr/local/tomcat/logs/

# 2.运行

docker-compose -f ./docker-compose-tomcat.yml up -d

-d：后台运行
-f：指定yml文件
